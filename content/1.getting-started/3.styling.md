## className Prop

Every shadcn-flutter component accepts a `className` prop that takes [Flutterwind](https://flutterwind.brighthustle.in) utility classes — Tailwind CSS syntax, but for Flutter:

```dart [example.dart]
ShadcnButton(
  className: 'rounded-full px-8',
  onPressed: () {},
  child: Text('Rounded Button'),
)
```

## className Extension

Any Flutter widget can also use the `.className()` extension method:

```dart [example.dart]
Container(
  child: Text('Hello'),
).className('bg-blue-500 text-white p-4 rounded-lg')
```

## Utility Classes

### Spacing

```dart
widget.className('p-4')        // padding all sides
widget.className('px-4 py-2')  // horizontal & vertical
widget.className('pt-2 pb-4')  // top & bottom
widget.className('m-4')        // margin all sides
widget.className('mx-auto')    // horizontal auto margin
```

### Colors

```dart
widget.className('bg-primary')           // theme primary
widget.className('bg-destructive/50')    // with 50% opacity
widget.className('text-muted-foreground') // muted text
widget.className('text-red-500')          // Tailwind color
```

### Borders

```dart
widget.className('border border-input rounded-md')
widget.className('rounded-full')
widget.className('border-2 border-red-500')
```

### Sizing

```dart
widget.className('w-full h-10')
widget.className('w-80 min-h-20')
widget.className('size-9')      // width + height
widget.className('max-w-sm')
```

### Shadows

```dart
widget.className('shadow-sm')
widget.className('shadow-md')
widget.className('shadow-lg')
```

### Focus Ring

```dart
widget.className('ring-[3px] ring-ring/20')
widget.className('ring-offset-2 ring-offset-background')
```

## Interactive Variants

Use variant prefixes for interactive styling:

### Hover

```dart
widget.className('hover:bg-primary/90')
```

### Focus Visible

```dart
widget.className('focus-visible:ring-ring/20 focus-visible:ring-[3px]')
```

### Active

```dart
widget.className('active:scale-95')
```

### Disabled

```dart
widget.className('disabled:opacity-50 disabled:pointer-events-none')
```

## Dark Mode

Use the `dark:` prefix to apply styles only in dark mode:

```dart
widget.className('bg-white dark:bg-zinc-900 text-black dark:text-white')
```

## Arbitrary Values

Use square brackets for custom values not covered by the preset scale:

```dart
widget.className('w-[320px] h-[48px] bg-[#FF5733] rounded-[12px]')
```

## Combining with Component Variants

The `className` prop merges with the component's built-in variant styles:

```dart [example.dart]
// The button already has 'outline' variant styles —
// className adds on top of them
ShadcnButton(
  variant: 'outline',
  className: 'border-2 border-blue-500 text-blue-500',
  onPressed: () {},
  child: Text('Custom Outline'),
)
```

## Class Variance Authority (CVA)

shadcn-flutter includes a Dart port of [CVA](https://cva.style/) for building your own variant-driven components:

```dart [example.dart]
final alertConfig = CvaConfig(
  base: 'rounded-lg border p-4',
  variants: {
    'variant': {
      'default': 'bg-background text-foreground',
      'destructive': 'border-destructive/50 text-destructive',
    },
  },
  defaultVariants: {'variant': 'default'},
);

// Usage
final classes = cva(alertConfig, values: {'variant': 'destructive'});
// → 'rounded-lg border p-4 border-destructive/50 text-destructive'
```
