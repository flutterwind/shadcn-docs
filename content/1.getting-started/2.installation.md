## Prerequisites

- Flutter SDK 3.10 or higher
- Dart SDK 3.0 or higher

## 1. Add the package

:::code-group
```sh [terminal]
flutter pub add shadcn_flutter
```
:::

Or add it manually to your `pubspec.yaml`:

```yaml [pubspec.yaml]
dependencies:
  shadcn_flutter:
    git:
      url: https://github.com/flutterwind/shadcn_flutter.git
      path: packages/shadcn_flutter
```

Then run:

```sh [terminal]
flutter pub get
```

## 2. Set up ShadcnFlutterApp

The easiest way to get started is to use `ShadcnFlutterApp` — it wraps your app with FlutterWind and the shadcn theme automatically:

```dart [main.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

void main() {
  runApp(
    ShadcnFlutterApp(
      title: 'My App',
      themeMode: ThemeMode.system,
      home: const HomePage(),
    ),
  );
}
```

### ShadcnFlutterApp props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `home` | `Widget` | — | Root widget (required) |
| `title` | `String` | `'shadcn_flutter app'` | App title |
| `themeMode` | `ThemeMode` | `ThemeMode.system` | Light / dark / system |
| `lightTheme` | `ShadcnThemeData?` | `null` | Custom light theme |
| `darkTheme` | `ShadcnThemeData?` | `null` | Custom dark theme |
| `showDevTools` | `bool` | `false` | FlutterWind devtools overlay |
| `loadingWidget` | `Widget?` | `null` | Shown during init |
| `onInitComplete` | `VoidCallback?` | `null` | Called when ready |

## 3. Manual setup (advanced)

If you need more control, wrap your `MaterialApp` with `FlutterWind` and `ShadcnTheme` manually:

```dart [main.dart]
import 'package:flutter/material.dart';
import 'package:flutterwind_core/flutterwind.dart';
import 'package:shadcn_flutter/shadcn_flutter.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return FlutterWind(
      child: ShadcnTheme(
        data: ShadcnThemeData.light(),
        child: MaterialApp(
          title: 'My App',
          home: const HomePage(),
        ),
      ),
    );
  }
}
```

## 4. Start using components

```dart [home_page.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

class HomePage extends StatelessWidget {
  const HomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            ShadcnButton(
              onPressed: () {},
              child: const Text('Click me'),
            ),
            const SizedBox(height: 16),
            ShadcnInput(
              hintText: 'Enter your email',
              type: 'email',
            ).className('w-80'),
          ],
        ),
      ),
    );
  }
}
```

## 5. Theme configuration (optional)

Create a `tailwind.config.yaml` at your project root for color overrides:

```yaml [tailwind.config.yaml]
theme:
  colors:
    primary: "#18181B"
    primary-foreground: "#FAFAFA"
    secondary: "#F4F4F5"
    secondary-foreground: "#18181B"
    destructive: "#EF4444"
    destructive-foreground: "#FAFAFA"
    background: "#FFFFFF"
    foreground: "#09090B"
    border: "#E4E4E7"
    input: "#E4E4E7"
    ring: "#18181B"
```
