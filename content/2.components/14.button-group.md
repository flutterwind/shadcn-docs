---
title: Button Group
description: Group buttons horizontally or vertically with merged borders.
icon: 'lucide:columns'
---

## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

ButtonGroup(
  children: [
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Left')),
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Center')),
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Right')),
  ],
)
```

Buttons inside a group automatically get merged borders — first and last buttons keep their outer rounded corners, while middle buttons have flat edges.

## Vertical

```dart [example.dart]
ButtonGroup(
  orientation: Axis.vertical,
  children: [
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Top')),
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Middle')),
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Bottom')),
  ],
)
```

## With Separator

```dart [example.dart]
ButtonGroup(
  children: [
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Save')),
    ButtonGroupSeparator(),
    ShadcnButton(
      variant: 'outline',
      size: 'icon',
      onPressed: () {},
      child: Icon(Icons.arrow_drop_down),
    ),
  ],
)
```

## With Text

```dart [example.dart]
ButtonGroup(
  children: [
    ShadcnButton(variant: 'outline', size: 'icon-sm', onPressed: () {}, child: Icon(Icons.chevron_left)),
    ButtonGroupText(child: Text('Page 1 of 10')),
    ShadcnButton(variant: 'outline', size: 'icon-sm', onPressed: () {}, child: Icon(Icons.chevron_right)),
  ],
)
```

## API Reference

### ButtonGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `List<Widget>` | — | Group children (required) |
| `orientation` | `Axis` | `Axis.horizontal` | Layout direction |
| `className` | `String?` | `null` | Flutterwind utility classes |

### ButtonGroupSeparator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `String?` | `null` | Flutterwind utility classes |
| `orientation` | `Axis?` | `null` | Override direction (auto-detected from group) |

### ButtonGroupText

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `child` | `Widget` | — | Text content (required) |
| `className` | `String?` | `null` | Flutterwind utility classes |
