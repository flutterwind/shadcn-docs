## Usage

Use `ShadcnDialog.show()` to display a modal dialog:

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

ShadcnDialog.show(
  context: context,
  title: Text('Are you sure?'),
  description: Text('This action cannot be undone.'),
  content: Text('This will permanently delete your account and all associated data.'),
  confirmLabel: 'Delete',
  cancelLabel: 'Cancel',
  onConfirm: () {
    // Handle deletion
  },
);
```

## Alert Dialog

```dart [example.dart]
ShadcnDialog.show(
  context: context,
  title: Text('Success'),
  content: Text('Your changes have been saved.'),
  confirmLabel: 'OK',
);
```

## Custom Content

```dart [example.dart]
ShadcnDialog.show(
  context: context,
  title: Text('Edit Profile'),
  description: Text('Update your account information.'),
  content: Column(
    children: [
      ShadcnInput(hintText: 'Name'),
      SizedBox(height: 12),
      ShadcnInput(hintText: 'Email', type: 'email'),
    ],
  ),
  confirmLabel: 'Save',
  cancelLabel: 'Cancel',
  onConfirm: () {
    // Save changes
  },
);
```

## API Reference

### ShadcnDialog.show()

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `context` | `BuildContext` | — | Build context (required) |
| `title` | `Widget` | — | Dialog title (required) |
| `description` | `Widget?` | `null` | Subtitle / description |
| `content` | `Widget` | — | Dialog body (required) |
| `confirmLabel` | `String?` | `null` | Confirm button text |
| `cancelLabel` | `String?` | `null` | Cancel button text |
| `onConfirm` | `VoidCallback?` | `null` | Confirm callback |

::alert{type="info"}
The dialog is rendered inside a `ShadcnCard` with a max width of 420px and a 60% black barrier overlay. Cancel automatically pops the dialog. Confirm fires `onConfirm` then pops.
::
