## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

ShadcnInput(
  hintText: 'Email',
  type: 'email',
  onChanged: (value) {
    print(value);
  },
)
```

## Input Types

### Text (default)

```dart [example.dart]
ShadcnInput(
  hintText: 'Enter text',
)
```

### Password

```dart [example.dart]
ShadcnInput(
  type: 'password',
  hintText: 'Password',
)
```

### Email

```dart [example.dart]
ShadcnInput(
  type: 'email',
  hintText: 'you@example.com',
)
```

### Number

```dart [example.dart]
ShadcnInput(
  type: 'number',
  hintText: '0',
)
```

### Phone

```dart [example.dart]
ShadcnInput(
  type: 'tel',
  hintText: '+1 (555) 000-0000',
)
```

### URL

```dart [example.dart]
ShadcnInput(
  type: 'url',
  hintText: 'https://example.com',
)
```

### File

Renders a native file picker button instead of a text field:

```dart [example.dart]
ShadcnInput(
  type: 'file',
  accept: 'image/*',
  allowMultiple: true,
  onFilesSelected: (files) {
    for (final file in files) {
      print(file.name);
    }
  },
)
```

The `accept` prop supports MIME-type filters like `'image/*'`, `'video/*'`, `'audio/*'`, `'application/pdf'`.

## States

### Disabled

```dart [example.dart]
ShadcnInput(
  hintText: 'Disabled',
  enabled: false,
)
```

### Read Only

```dart [example.dart]
ShadcnInput(
  readOnly: true,
  initialValue: 'Cannot edit this',
)
```

### Invalid

```dart [example.dart]
ShadcnInput(
  hintText: 'Required field',
  invalid: true,
)
```

## With Custom Width

```dart [example.dart]
ShadcnInput(
  hintText: 'Full width input',
).className('w-full')
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `controller` | `TextEditingController?` | `null` | Text controller |
| `hintText` | `String?` | `null` | Placeholder text |
| `initialValue` | `String?` | `null` | Initial text value |
| `onChanged` | `ValueChanged<String>?` | `null` | Text change callback |
| `onSubmitted` | `ValueChanged<String>?` | `null` | Submit callback |
| `enabled` | `bool` | `true` | Whether input is enabled |
| `readOnly` | `bool` | `false` | Read-only mode |
| `focusNode` | `FocusNode?` | `null` | Focus node |
| `className` | `String?` | `null` | Flutterwind utility classes |
| `invalid` | `bool` | `false` | Error / invalid state |
| `type` | `String` | `'text'` | Input type |
| `maxLines` | `int?` | `1` | Maximum lines |
| `minLines` | `int?` | `null` | Minimum lines |

### File-specific Props

These only apply when `type: 'file'`:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onFilesSelected` | `ValueChanged<List<PlatformFile>>?` | `null` | File selection callback |
| `allowedExtensions` | `List<String>?` | `null` | Allowed file extensions |
| `allowMultiple` | `bool` | `false` | Allow picking multiple files |
| `accept` | `String?` | `null` | MIME type filter |
