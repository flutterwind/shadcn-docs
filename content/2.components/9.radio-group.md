## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

String? selected;

ShadcnRadioGroup<String>(
  value: selected,
  onChanged: (value) {
    setState(() => selected = value);
  },
  options: [
    ShadcnRadioOption(value: 'default', label: 'Default'),
    ShadcnRadioOption(value: 'comfortable', label: 'Comfortable'),
    ShadcnRadioOption(value: 'compact', label: 'Compact'),
  ],
)
```

## With Enum Values

```dart [example.dart]
enum Theme { light, dark, system }

ShadcnRadioGroup<Theme>(
  value: selectedTheme,
  onChanged: (value) {
    setState(() => selectedTheme = value);
  },
  options: [
    ShadcnRadioOption(value: Theme.light, label: 'Light'),
    ShadcnRadioOption(value: Theme.dark, label: 'Dark'),
    ShadcnRadioOption(value: Theme.system, label: 'System'),
  ],
)
```

## API Reference

### ShadcnRadioGroup\<T\>

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `T?` | — | Selected value (required) |
| `onChanged` | `ValueChanged<T?>` | — | Change callback (required) |
| `options` | `List<ShadcnRadioOption<T>>` | — | Radio options (required) |

### ShadcnRadioOption\<T\>

| Prop | Type | Description |
|------|------|-------------|
| `value` | `T` | Option value |
| `label` | `String` | Display label |
