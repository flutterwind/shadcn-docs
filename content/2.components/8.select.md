## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

String? selected;

ShadcnSelect<String>(
  value: selected,
  hint: 'Select a framework',
  onChanged: (value) {
    setState(() => selected = value);
  },
  items: [
    DropdownMenuItem(value: 'flutter', child: Text('Flutter')),
    DropdownMenuItem(value: 'react', child: Text('React Native')),
    DropdownMenuItem(value: 'swiftui', child: Text('SwiftUI')),
    DropdownMenuItem(value: 'jetpack', child: Text('Jetpack Compose')),
  ],
)
```

## With Hint

```dart [example.dart]
ShadcnSelect<String>(
  value: null,
  hint: 'Choose an option...',
  onChanged: (value) {},
  items: [
    DropdownMenuItem(value: 'a', child: Text('Option A')),
    DropdownMenuItem(value: 'b', child: Text('Option B')),
    DropdownMenuItem(value: 'c', child: Text('Option C')),
  ],
)
```

## With Integer Values

```dart [example.dart]
ShadcnSelect<int>(
  value: selectedAge,
  hint: 'Select age group',
  onChanged: (value) {
    setState(() => selectedAge = value);
  },
  items: [
    DropdownMenuItem(value: 18, child: Text('18-24')),
    DropdownMenuItem(value: 25, child: Text('25-34')),
    DropdownMenuItem(value: 35, child: Text('35-44')),
  ],
)
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `T?` | — | Currently selected value (required) |
| `onChanged` | `ValueChanged<T?>` | — | Change callback (required) |
| `items` | `List<DropdownMenuItem<T>>` | — | Dropdown items (required) |
| `hint` | `String?` | `null` | Placeholder hint text |
