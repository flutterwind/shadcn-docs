## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

ShadcnCard(
  child: Column(
    crossAxisAlignment: CrossAxisAlignment.start,
    children: [
      Text('Card Title',
        style: TextStyle(fontSize: 18, fontWeight: FontWeight.w600)),
      SizedBox(height: 8),
      Text('Card description with some content here.'),
    ],
  ),
)
```

## Custom Padding

```dart [example.dart]
ShadcnCard(
  padding: EdgeInsets.all(24),
  child: Text('More padding'),
)
```

## Custom Styling

```dart [example.dart]
ShadcnCard(
  className: 'bg-zinc-50 dark:bg-zinc-900 shadow-lg',
  child: Text('Custom background & shadow'),
)
```

## With Other Components

```dart [example.dart]
ShadcnCard(
  child: Column(
    crossAxisAlignment: CrossAxisAlignment.start,
    children: [
      Text('Create Project', style: TextStyle(fontSize: 18, fontWeight: FontWeight.w600)),
      SizedBox(height: 4),
      Text('Deploy your new project in one click.'),
      SizedBox(height: 16),
      ShadcnInput(hintText: 'Project name'),
      SizedBox(height: 12),
      ShadcnButton(
        onPressed: () {},
        child: Text('Create'),
      ),
    ],
  ),
)
```
