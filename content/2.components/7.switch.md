## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

bool isEnabled = false;

ShadcnSwitch(
  value: isEnabled,
  onChanged: (value) {
    setState(() => isEnabled = value);
  },
)
```

## With Label

```dart [example.dart]
ShadcnSwitch(
  value: isEnabled,
  onChanged: (value) {
    setState(() => isEnabled = value);
  },
  label: 'Airplane mode',
)
```

## Settings List Example

```dart [example.dart]
Column(
  children: [
    ShadcnSwitch(value: darkMode, onChanged: (v) => setState(() => darkMode = v), label: 'Dark mode'),
    SizedBox(height: 12),
    ShadcnSwitch(value: notifications, onChanged: (v) => setState(() => notifications = v), label: 'Notifications'),
    SizedBox(height: 12),
    ShadcnSwitch(value: analytics, onChanged: (v) => setState(() => analytics = v), label: 'Analytics'),
  ],
)
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `bool` | — | Whether on (required) |
| `onChanged` | `ValueChanged<bool>` | — | Change callback (required) |
| `label` | `String?` | `null` | Optional label text |
