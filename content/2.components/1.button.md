## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

ShadcnButton(
  onPressed: () {},
  child: Text('Button'),
)
```

## Sizes

| Size | Dimensions | Description |
|------|-----------|-------------|
| `'default'` | `h-9 px-4 py-2` | Standard button |
| `'xs'` | `h-6 px-2` | Extra small |
| `'sm'` | `h-8 px-3` | Small |
| `'lg'` | `h-10 px-8` | Large |
| `'icon'` | `size-9` | Square icon button |
| `'icon-xs'` | `size-6` | Icon extra small |
| `'icon-sm'` | `size-8` | Icon small |
| `'icon-lg'` | `size-10` | Icon large |

```dart [example.dart]
ShadcnButton(
  size: 'sm',
  onPressed: () {},
  child: Text('Small'),
)
```

## Icon Button

```dart [example.dart]
ShadcnButton(
  size: 'icon',
  variant: 'outline',
  onPressed: () {},
  child: Icon(Icons.settings),
)
```

## With Custom Class

```dart [example.dart]
ShadcnButton(
  className: 'rounded-full px-8',
  onPressed: () {},
  child: Text('Rounded'),
)
```

## Invalid State

When `invalid` is `true`, the button renders with a destructive focus ring:

```dart [example.dart]
ShadcnButton(
  invalid: true,
  onPressed: () {},
  child: Text('Error'),
)
```

## Inside a Button Group

Buttons automatically adjust their border radius when placed inside a [`ButtonGroup`](/components/button-group):

```dart [example.dart]
ButtonGroup(
  children: [
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Left')),
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Center')),
    ShadcnButton(variant: 'outline', onPressed: () {}, child: Text('Right')),
  ],
)
```
