---

## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

bool isChecked = false;

ShadcnCheckbox(
  value: isChecked,
  onChanged: (value) {
    setState(() => isChecked = value);
  },
)
```

## With Label

```dart [example.dart]
ShadcnCheckbox(
  value: isChecked,
  onChanged: (value) {
    setState(() => isChecked = value);
  },
  label: 'Accept terms and conditions',
)
```

## In a Form

```dart [example.dart]
Column(
  crossAxisAlignment: CrossAxisAlignment.start,
  children: [
    ShadcnCheckbox(
      value: notifications,
      onChanged: (v) => setState(() => notifications = v),
      label: 'Email notifications',
    ),
    SizedBox(height: 8),
    ShadcnCheckbox(
      value: marketing,
      onChanged: (v) => setState(() => marketing = v),
      label: 'Marketing emails',
    ),
  ],
)
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `bool` | — | Whether checked (required) |
| `onChanged` | `ValueChanged<bool>` | — | Change callback (required) |
| `label` | `String?` | `null` | Optional label text |
