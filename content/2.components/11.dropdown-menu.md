## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

ShadcnDropdownMenu(
  trigger: ShadcnButton(
    variant: 'outline',
    onPressed: null,
    child: Text('Open Menu'),
  ),
  items: [
    ShadcnDropdownItem(
      label: 'Edit',
      onTap: () => print('Edit'),
    ),
    ShadcnDropdownItem(
      label: 'Duplicate',
      onTap: () => print('Duplicate'),
    ),
    ShadcnDropdownItem(
      label: 'Delete',
      onTap: () => print('Delete'),
    ),
  ],
)
```

## With Icon Trigger

```dart [example.dart]
ShadcnDropdownMenu(
  trigger: ShadcnButton(
    variant: 'ghost',
    size: 'icon',
    onPressed: null,
    child: Icon(Icons.more_vert),
  ),
  items: [
    ShadcnDropdownItem(label: 'Profile', onTap: () {}),
    ShadcnDropdownItem(label: 'Settings', onTap: () {}),
    ShadcnDropdownItem(label: 'Sign out', onTap: () {}),
  ],
)
```

## API Reference

### ShadcnDropdownMenu

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `trigger` | `Widget` | — | Widget that opens the menu (required) |
| `items` | `List<ShadcnDropdownItem>` | — | Menu items (required) |

### ShadcnDropdownItem

| Prop | Type | Description |
|------|------|-------------|
| `label` | `String` | Menu item text |
| `onTap` | `VoidCallback` | Tap callback |
