---
title: Input Group
description: Group inputs, buttons, addons, and text into a combined control.
icon: 'lucide:group'
---

## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

InputGroup(
  children: [
    InputGroupText(child: Text('https://')),
    InputGroupInput(hintText: 'example.com'),
  ],
)
```

## With Search Button

```dart [example.dart]
InputGroup(
  children: [
    InputGroupInput(hintText: 'Search...'),
    InputGroupButton(
      onPressed: () {},
      child: Icon(Icons.search),
    ),
  ],
)
```

## With Addon

```dart [example.dart]
InputGroup(
  children: [
    InputGroupAddon(
      align: InputGroupAddonAlign.inlineStart,
      children: [Icon(Icons.email, size: 16)],
    ),
    InputGroupInput(
      hintText: 'you@example.com',
      type: 'email',
    ),
  ],
)
```

## With Textarea

```dart [example.dart]
InputGroup(
  children: [
    InputGroupText(child: Text('Message')),
    InputGroupTextarea(
      hintText: 'Type your message...',
      minLines: 3,
    ),
  ],
)
```

## End Addon

```dart [example.dart]
InputGroup(
  children: [
    InputGroupInput(hintText: '0.00'),
    InputGroupAddon(
      align: InputGroupAddonAlign.inlineEnd,
      children: [Text('USD')],
    ),
  ],
)
```

## API Reference

### InputGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `List<Widget>` | — | Group items (required) |
| `className` | `String?` | `null` | Flutterwind utility classes |

### InputGroupInput

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `controller` | `TextEditingController?` | `null` | Text controller |
| `hintText` | `String?` | `null` | Placeholder text |
| `onChanged` | `ValueChanged<String>?` | `null` | Change callback |
| `enabled` | `bool` | `true` | Whether enabled |
| `focusNode` | `FocusNode?` | `null` | Focus node |
| `className` | `String?` | `null` | Flutterwind classes |
| `invalid` | `bool` | `false` | Error state |
| `type` | `String` | `'text'` | Input type |

### InputGroupTextarea

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `controller` | `TextEditingController?` | `null` | Text controller |
| `hintText` | `String?` | `null` | Placeholder text |
| `onChanged` | `ValueChanged<String>?` | `null` | Change callback |
| `enabled` | `bool` | `true` | Whether enabled |
| `focusNode` | `FocusNode?` | `null` | Focus node |
| `className` | `String?` | `null` | Flutterwind classes |
| `invalid` | `bool` | `false` | Error state |
| `maxLines` | `int?` | `null` | Maximum lines |
| `minLines` | `int?` | `null` | Minimum lines |

### InputGroupButton

| Prop | Type | Default | Description |
