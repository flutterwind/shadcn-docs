## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

ShadcnBadge(
  child: Text('Badge'),
)
```

## Variants

### Default

```dart [example.dart]
ShadcnBadge(
  variant: 'default',
  child: Text('Default'),
)
```

### Secondary

```dart [example.dart]
ShadcnBadge(
  variant: 'secondary',
  child: Text('Secondary'),
)
```

### Outline

```dart [example.dart]
ShadcnBadge(
  variant: 'outline',
  child: Text('Outline'),
)
```

### Destructive

```dart [example.dart]
ShadcnBadge(
  variant: 'destructive',
  child: Text('Destructive'),
)
```

## Custom Styling

```dart [example.dart]
ShadcnBadge(
  className: 'rounded-full px-4',
  child: Text('Custom'),
)
```
