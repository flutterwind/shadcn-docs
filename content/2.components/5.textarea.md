## Usage

```dart [example.dart]
import 'package:shadcn_flutter/shadcn_flutter.dart';

ShadcnTextarea(
  hintText: 'Type your message here.',
  onChanged: (value) {
    print(value);
  },
)
```

## Custom Line Limits

```dart [example.dart]
ShadcnTextarea(
  hintText: 'Short message',
  minLines: 2,
  maxLines: 4,
)
```

## States

### Disabled

```dart [example.dart]
ShadcnTextarea(
  hintText: 'Disabled',
  enabled: false,
)
```

### Read Only

```dart [example.dart]
ShadcnTextarea(
  readOnly: true,
)
```

### Invalid

```dart [example.dart]
ShadcnTextarea(
  hintText: 'Required field',
  invalid: true,
)
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `controller` | `TextEditingController?` | `null` | Text controller |
| `hintText` | `String?` | `null` | Placeholder text |
| `onChanged` | `ValueChanged<String>?` | `null` | Text change callback |
| `minLines` | `int?` | `3` | Minimum visible lines |
| `maxLines` | `int?` | `6` | Maximum visible lines |
| `focusNode` | `FocusNode?` | `null` | Focus node |
| `className` | `String?` | `null` | Flutterwind utility classes |
| `invalid` | `bool` | `false` | Error / invalid state |
| `enabled` | `bool` | `true` | Whether enabled |
| `readOnly` | `bool` | `false` | Read-only mode |
