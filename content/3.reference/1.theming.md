shadcn-flutter has its own design-token system built on top of [Flutterwind](https://flutterwind.brighthustle.in). Every component reads colors and radii from `ShadcnThemeData`, so a single token change updates every button, input, card, and badge at once.

## Quick Start

The simplest setup — `ShadcnFlutterApp` gives you light and dark themes with zero config:

```dart
ShadcnFlutterApp(
  themeMode: ThemeMode.system,
  home: const HomePage(),
)
```

## Built-in Themes

shadcn-flutter ships with two pre-built themes based on the zinc color palette:

```dart
ShadcnThemeData.light()  // zinc-based light theme
ShadcnThemeData.dark()   // zinc-based dark theme
```

## Custom Theme

Override any token by passing a custom `ShadcnThemeData`:

```dart
ShadcnFlutterApp(
  lightTheme: ShadcnThemeData(
    primary: Color(0xFF2563EB),
    primaryForeground: Color(0xFFFFFFFF),
    secondary: Color(0xFFF1F5F9),
    secondaryForeground: Color(0xFF0F172A),
    destructive: Color(0xFFEF4444),
    destructiveForeground: Color(0xFFFFFFFF),
    background: Color(0xFFFFFFFF),
    foreground: Color(0xFF0F172A),
    card: Color(0xFFFFFFFF),
    cardForeground: Color(0xFF0F172A),
    border: Color(0xFFE2E8F0),
    input: Color(0xFFE2E8F0),
    muted: Color(0xFFF1F5F9),
    mutedForeground: Color(0xFF64748B),
    accent: Color(0xFFF1F5F9),
    accentForeground: Color(0xFF0F172A),
    radiusSm: 6.0,
    radiusMd: 8.0,
    radiusLg: 12.0,
    spacingUnit: 4.0,
  ),
  darkTheme: ShadcnThemeData.dark(),
  home: const HomePage(),
)
```

## Theme Tokens Reference

### Color Tokens

| Token | Usage |
|-------|-------|
| `primary` | Button defaults, checkbox fill, switch active track |
| `primaryForeground` | Text on primary backgrounds |
| `secondary` | Secondary button and badge backgrounds |
| `secondaryForeground` | Text on secondary backgrounds |
| `destructive` | Destructive buttons, error states |
| `destructiveForeground` | Text on destructive backgrounds |
| `background` | Page and app backgrounds |
| `foreground` | Default text color |
| `card` | Card component background |
| `cardForeground` | Card text color |
| `border` | Border color for cards, checkboxes |
| `input` | Input component border color |
| `muted` | Muted backgrounds like tab bars |
| `mutedForeground` | Muted text for placeholders, descriptions |
| `accent` | Hover backgrounds for ghost and outline variants |
| `accentForeground` | Text on accent backgrounds |

### Radius & Spacing Tokens

| Token | Default | Usage |
|-------|---------|-------|
| `radiusSm` | `8.0` | Small radius — badges |
| `radiusMd` | `10.0` | Medium radius — inputs, selects, dropdowns |
| `radiusLg` | `12.0` | Large radius — cards, dialogs |
| `spacingUnit` | `4.0` | Base multiplier for spacing utilities |

## Accessing the Theme in Code

Use `ShadcnTheme.of(context)` to read tokens anywhere:

```dart
final theme = ShadcnTheme.of(context);

Container(
  decoration: BoxDecoration(
    color: theme.card,
    border: Border.all(color: theme.border),
    borderRadius: BorderRadius.circular(theme.radiusMd),
  ),
  child: Text(
    'Themed container',
    style: TextStyle(color: theme.foreground),
  ),
)
```

## Dark Mode

`ShadcnFlutterApp` respects Flutter's `ThemeMode`. Components automatically switch themes:

```dart
ShadcnFlutterApp(
  themeMode: ThemeMode.system,
  lightTheme: ShadcnThemeData.light(),
  darkTheme: ShadcnThemeData.dark(),
  home: const HomePage(),
)
```

You can also use Flutterwind's `dark:` variant prefix for manual dark mode overrides:

```dart
widget.className('bg-white dark:bg-zinc-950 text-black dark:text-white')
```

::alert{type="info" icon="lucide:info"}
See the [Flutterwind theming docs](https://flutterwind.brighthustle.in/getting-started/theming) for more details on `ThemeMode` and `dark:` variant usage.
::

## Tailwind Config File

You can also define color tokens via a `tailwind.config.yaml` file in your project root. Flutterwind will merge these values at startup:

```yaml
theme:
  colors:
    primary: "#18181B"
    primary-foreground: "#FAFAFA"
    secondary: "#F4F4F5"
    secondary-foreground: "#18181B"
    destructive: "#DC2626"
    destructive-foreground: "#FAFAFA"
    background: "#FFFFFF"
    foreground: "#09090B"
    border: "#E4E4E7"
    input: "#E4E4E7"
    ring: "#18181B"
    muted: "#F4F4F5"
    muted-foreground: "#71717A"
    accent: "#F4F4F5"
    accent-foreground: "#18181B"
```

::alert{type="info" icon="lucide:link"}
For full configuration options (colors with shades, spacing scales, breakpoints, typography, shadows, filters), see the [Flutterwind Configuration docs](https://flutterwind.brighthustle.in/getting-started/configuration).
::
