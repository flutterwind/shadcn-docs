## What is Flutterwind?

[Flutterwind](https://flutterwind.brighthustle.in) is a Tailwind CSS utility engine for Flutter. It lets you style widgets using familiar utility class strings instead of verbose `BoxDecoration`, `EdgeInsets`, and `TextStyle` code.

shadcn-flutter uses Flutterwind under the hood for **all** component styling. Every component accepts a `className` prop for customization.

## Setup

`ShadcnFlutterApp` wraps your app with `FlutterWind` automatically. If using the manual setup, ensure `FlutterWind` is an ancestor:

```dart [example.dart]
import 'package:flutterwind_core/flutterwind.dart';

FlutterWind(
  child: MaterialApp(...),
)
```

## className Extension

All widgets get a `.className()` extension method:

```dart [example.dart]
Container(
  child: Text('Hello'),
).className('bg-blue-500 text-white p-4 rounded-lg')
```

## Utility Reference

### Spacing

| Class | Effect |
|-------|--------|
| `p-{n}` | Padding all sides |
| `px-{n}`, `py-{n}` | Horizontal / vertical padding |
| `pt-{n}`, `pr-{n}`, `pb-{n}`, `pl-{n}` | Individual sides |
| `m-{n}` | Margin all sides |
| `mx-{n}`, `my-{n}` | Horizontal / vertical margin |

### Sizing

| Class | Effect |
|-------|--------|
| `w-{n}`, `h-{n}` | Width / height |
| `w-full`, `h-full` | 100% |
| `size-{n}` | Both width and height |
| `min-w-{n}`, `min-h-{n}` | Minimum constraints |
| `max-w-{n}`, `max-h-{n}` | Maximum constraints |

### Colors

| Class | Effect |
|-------|--------|
| `bg-{color}` | Background color |
| `text-{color}` | Text color |
| `border-{color}` | Border color |
| `bg-{color}/{opacity}` | With opacity (0â€“100) |

### Borders

| Class | Effect |
|-------|--------|
| `border` | 1px border |
| `border-{n}` | Custom width border |
| `rounded-{size}` | Border radius (`sm`, `md`, `lg`, `xl`, `full`) |
| `ring-{n}` | Focus ring width |
| `ring-{color}` | Ring color |
| `ring-offset-{n}` | Ring offset width |
| `ring-offset-{color}` | Ring offset color |

### Typography

| Class | Effect |
|-------|--------|
| `text-xs` through `text-9xl` | Font size |
| `font-bold`, `font-semibold`, etc. | Font weight |
| `leading-{n}` | Line height |
| `tracking-{n}` | Letter spacing |

### Shadows

| Class | Effect |
|-------|--------|
| `shadow-xs`, `shadow-sm` | Subtle shadows |
| `shadow`, `shadow-md` | Medium shadows |
| `shadow-lg`, `shadow-xl`, `shadow-2xl` | Large shadows |

## Interactive Variants

| Prefix | Trigger |
|--------|---------|
| `hover:` | Mouse hover |
| `focus-visible:` | Keyboard focus |
| `active:` | Active / pressed |
| `disabled:` | Disabled state |
| `dark:` | Dark mode |

```dart [example.dart]
widget.className('hover:bg-primary/90 focus-visible:ring-[3px] active:scale-95 disabled:opacity-50')
```

## Arbitrary Values

Use square brackets for one-off values:

```dart [example.dart]
widget.className('w-[320px] h-[48px] bg-[#FF5733] rounded-[12px] ring-[3px]')
```

## shadcn Semantic Classes

The `ShadcnUtilitiesPlugin` registers additional utility classes that map to theme tokens:

| Class | Maps to |
|-------|---------|
| `shadcn-surface` | `ShadcnTheme.background` |
| `shadcn-muted` | `ShadcnTheme.muted` |
| `shadcn-primary` | `ShadcnTheme.primary` |

These are auto-installed when using any shadcn-flutter component.
